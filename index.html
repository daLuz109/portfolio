Project Summary

  • ImunoCell

    This is a highscool project. The idea is that you are a white cell and have to fight against invasors like viruses and bacteria.

    Part of the project was to show how games can help you learn, and for that reason, I wanted to make it very accurate.

    The overall feel of the game is inspired by games like Hotline Miami, so it feels very smooth.

    This project gave me a lot of experience on Gamemaker Studio, but mostly on game design and art.

  • Catwitch

    My favorite project! This is a story-focused game with a bit of horror mixed in. You are a cat who's also a witch and you have to hunt for ingredients and make potions for the dog soldiers.

    I decided to limit the art to only 4 colors, mostly because I think it looks cool, but also because I wanted to see what I could do with limited resources.

    This project, again, gave me a lot of experience on Gamemaker, game design and art, but I also made music for this one.

  • Cleavage

    This one is a Minecraft mod, a bit unconventional for a portfolio but I think it's important to be shown.

    On this mod you have a bunch of different gems which can be crafted using a new ore, and each one of them has its own set of 3 abilities.

    The main reason of putting this here is because I learned a lot about java while making it.
